# 🎭 Playwright Automation

Simple automation testing project using Playwright.

## 📦 Installation

Clone repository:

```
git clone https://github.com/amalliarizkii/Playwright.git
cd Playwright
```

Install dependencies:

```
npm install
```

Install Playwright browsers:

```
npx playwright install
```

## ▶️ Run Test

Run all tests:

```
npx playwright test
```

Run with UI (browser terlihat):

```
npx playwright test --headed
```

## 📁 Project Structure

```
tests/                # test files
playwright.config.ts  # configuration
```

## 🛠️ Tech Stack

* Playwright
* TypeScript / JavaScript
* Node.js

This project is for learning and practice automation testing using Playwright.
